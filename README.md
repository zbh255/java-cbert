# java-cbert
The java-cbert is java implement a cbert client

#### Usage

Mavan

>  in pom.xml add config item

```xml
<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://www.jitpack.io</url>
		</repository>
</repositories>
```

```xml
<dependency>
	    <groupId>com.github.zbh255</groupId>
	    <artifactId>java-cbert</artifactId>
	    <version>v0.1.0</version>
</dependency>
```
